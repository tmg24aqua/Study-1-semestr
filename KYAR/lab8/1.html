CSS (англ. Cascading Style Sheets «каскадные таблицы стилей») — формальный язык декодирования и описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML). Также может применяться к любым XML-документам, например, к SVG или XUL. Свойства CSS (Каскадные таблицы стилей) — это параметры, которые используются для задания внешнего вида и формирования элементов на веб-странице. Каждое свойство CSS описывает конкретный аспект стиля или элемента поведения. Свойства применяются к элементам через CSS-правила, составляющие из селектора, свойств и значений.
